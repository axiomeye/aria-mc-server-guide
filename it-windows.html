← Guide Guida Italiana Windows
Minecraft 1.20.1 Fabric ~15–20 min

Ciao! Questa guida ti aiuterà passo per passo a configurare Minecraft su Windows. Non serve essere esperti: ogni passaggio è spiegato nel dettaglio.

01

Installa un Launcher Minecraft

Il launcher è il programma che avvia Minecraft. Scegline uno:

  • SKLauncher, consigliato se non hai acquistato Minecraft — scaricalo da skmedix.pl
  • Launcher Ufficiale Minecraft, se hai un account Microsoft o Mojang — scaricalo da minecraft.net

Dopo l'installazione, apri il launcher, seleziona la versione 1.20.1 e avviala almeno una volta. Puoi chiuderla subito appena compare il menu principale: serve solo per creare i file di gioco.

02

Installa Java

Java è un programma gratuito necessario per far funzionare Fabric.

  1. Vai su adoptium.net
  2. Clicca Latest LTS release per scaricare l'installer
  3. Apri il file scaricato e segui le istruzioni (clicca sempre Next fino alla fine)

Se durante l'installazione di Fabric non compare nessun errore riguardo a Java, puoi saltare questo passaggio: forse lo hai già installato.

03

Installa Fabric

Fabric è un componente che permette al gioco di caricare le mod del server. È obbligatorio per connettersi.

  1. Scarica il Fabric Installer (si tratta di un file .jar)
  2. Fai doppio clic sul file appena scaricato per aprirlo
  3. Nella finestra che compare, assicurati che la versione sia 1.20.1
  4. Clicca Install e aspetta che finisca
04

Installa le Mod

Le mod sono piccole estensioni richieste dal server — senza di esse non ti farà connettere.

  1. Premi i tasti Win + R sulla tastiera contemporaneamente
  2. Digita %appdata% e premi Invio
  3. Si apre una cartella: cerca e apri quella chiamata .minecraft
  4. Se esiste già una cartella chiamata mods, eliminala completamente (clic destro → Elimina)
  5. Crea una nuova cartella vuota chiamata mods
  6. Scarica mods.zip dalla cartella condivisa (richiede un account Google)
  7. Apri il file .zip, seleziona tutto (Ctrl + A) e trascinalo nella cartella mods
05 — opzionale

Installa i Resource Pack

I resource pack cambiano l'aspetto visivo del gioco. Non sono obbligatori, ma migliorano l'esperienza.

  1. Scarica resourcepacks.zip dalla cartella condivisa
  2. Torna nella cartella .minecraft (come al punto 4, passi 1–3)
  3. Crea la cartella resourcepacks se non esiste già
  4. Apri lo zip e trascina tutto il contenuto in .minecraft/resourcepacks
  5. In gioco, vai su Opzioni → Resource Pack e sposta i pack dalla colonna sinistra a quella destra
06 — opzionale

Installa gli Shader

Gli shader migliorano molto la grafica, ma richiedono un PC abbastanza potente. Se il tuo PC è datato, salta questo passaggio.

  1. Scarica Iris e mettilo nella cartella mods: iris-1.7.6+mc1.20.1.jar
  2. Crea la cartella shaderpacks dentro .minecraft se non esiste
  3. Scarica Complementary Unbound e mettilo in shaderpacks: ComplementaryUnbound_r5.5.1.zip
  4. In gioco, vai su Opzioni → Shader e seleziona il pacchetto
07

Avvia il Server

Il server si accende e si spegne dall'AriA Panel.

  1. Apri l'AriA Panel e clicca Start
  2. Aspetta qualche minuto: il pannello mostra lo stato in tempo reale
  3. Quando compare l'indirizzo IP il server è acceso e puoi connetterti

Il server si spegne in automatico quando non ci sono giocatori collegati. Non preoccuparti di spegnerlo manualmente.

08

Collegati al Server

  1. Apri il launcher e seleziona il profilo Fabric 1.20.1
  2. Avvia il gioco
  3. Nel menu principale, clicca Multigiocatore → Connessione Diretta
  4. Copia e incolla l'indirizzo IP mostrato nell'AriA Panel (esempio: 1.2.3.4:25565)
  5. Clicca Connetti e buon divertimento!

Problemi comuni

Problema Soluzione
Fabric non si apre Assicurati di aver installato Java (passo 2)
Il gioco crasha subito Elimina la cartella mods e riestrai da zero
Non vedo il profilo Fabric nel launcher Rifai il passo 3 e assicurati di selezionare la versione 1.20.1
Non riesco a connettermi Controlla che l'AriA Panel mostri l'IP: potrebbe essere ancora in avvio
