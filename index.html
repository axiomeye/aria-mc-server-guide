Ciao! Questa guida ti aiuterà passo per passo a configurare Minecraft su Windows. Non serve essere esperti: ogni passaggio è spiegato nel dettaglio.

01

Installa un Launcher Minecraft

Il launcher è il programma che avvia Minecraft. Scegline uno:

  • SKLauncher, consigliato se non hai acquistato Minecraft: skmedix.pl
  • Launcher Ufficiale Minecraft, se hai un account Microsoft o Mojang: minecraft.net

Dopo l'installazione, apri il launcher, seleziona la versione 1.20.1 e avviala almeno una volta. Puoi chiuderla subito appena compare il menu principale: serve solo per creare i file di gioco.

02

Installa Java

Java è un programma gratuito necessario per far funzionare Fabric.

  1. Vai su adoptium.net
  2. Clicca Latest LTS release per scaricare l'installer
  3. Apri il file scaricato e segui le istruzioni (clicca sempre Next)

Se durante l'installazione di Fabric non compare nessun errore riguardo a Java, puoi saltare questo passaggio: forse lo hai già installato.

03

Installa Fabric

Fabric è un componente che permette al gioco di caricare le mod. È obbligatorio per connettersi.

  1. Scarica il Fabric Installer (si tratta di un file .jar)
  2. Fai doppio clic sul file per aprirlo
  3. Assicurati che la versione sia 1.20.1
  4. Clicca Install e aspetta che finisca
04

Installa le Mod

Le mod richieste dal server: senza di esse non ti farà connettere.

  1. Premi Win + R, digita %appdata% e premi Invio
  2. Apri la cartella .minecraft
  3. Se esiste già una cartella mods, eliminala completamente (clic destro → Elimina)
  4. Crea una nuova cartella vuota chiamata mods
  5. Scarica mods.zip dalla cartella condivisa (richiede account Microsoft)
  6. Apri lo zip, seleziona tutto (Ctrl + A) e trascinalo nella cartella mods
05 · opzionale

Installa i Resource Pack

  1. Scarica resourcepacks.zip dalla cartella condivisa
  2. Torna in .minecraft e crea la cartella resourcepacks se non esiste
  3. Trascina tutto il contenuto dello zip in .minecraft/resourcepacks
  4. In gioco: Opzioni → Resource Pack, sposta i pack a destra per attivarli
06 · opzionale

Installa gli Shader

Richiedono un PC abbastanza potente. Se il tuo PC è datato, salta questo passaggio.

  1. Scarica Iris e mettilo nella cartella mods
  2. Crea la cartella shaderpacks dentro .minecraft
  3. Scarica Complementary Unbound e mettilo in shaderpacks
  4. In gioco: Opzioni → Shader, seleziona il pacchetto
07

Avvia il Server

  1. Apri l'AriA Panel e clicca Start
  2. Aspetta qualche minuto: il pannello mostra lo stato in tempo reale
  3. Quando compare l'IP il server è pronto

Il server si spegne in automatico quando non ci sono giocatori. Non preoccuparti di spegnerlo manualmente.

08

Collegati al Server

  1. Apri il launcher e seleziona il profilo Fabric 1.20.1
  2. Nel menu principale: Multigiocatore → Connessione Diretta
  3. Incolla l'IP mostrato nell'AriA Panel (es. 1.2.3.4:25565)
  4. Clicca Connetti e buon divertimento!

Hey! This guide walks you through setting up Minecraft on Windows step by step. No experience required.

01

Install a Minecraft Launcher

  • SKLauncher, recommended if you don't own Minecraft: skmedix.pl
  • Official Minecraft Launcher: minecraft.net

After installing, open the launcher, select version 1.20.1 and launch it once. Close it as soon as the main menu appears: this just creates the game files.

02

Install Java

  1. Go to adoptium.net
  2. Click Latest LTS release
  3. Open the installer and keep clicking Next

If the Fabric installer opens with no Java error, you can skip this step.

03

Install Fabric

  1. Download the Fabric Installer (.jar file)
  2. Double-click to open it
  3. Make sure the version is 1.20.1
  4. Click Install
04

Install the Mods

  1. Press Win + R, type %appdata%, press Enter
  2. Open the .minecraft folder
  3. If a mods folder exists, delete it entirely (right-click → Delete)
  4. Create a new empty mods folder
  5. Download mods.zip from the shared folder (Microsoft account required)
  6. Open the zip, select all (Ctrl + A) and drag into mods
05 · optional

Install Resource Packs

  1. Download resourcepacks.zip from the shared folder
  2. Create a resourcepacks folder inside .minecraft
  3. Drag the zip contents into it
  4. In-game: Options → Resource Packs, move packs to the right column
06 · optional

Install Shaders

Require a reasonably powerful PC. Skip if your PC is old.

  1. Download Iris and place it in mods
  2. Create a shaderpacks folder inside .minecraft
  3. Download Complementary Unbound and place it in shaderpacks
  4. In-game: Options → Shaders, select the pack
07

Start the Server

  1. Open the AriA Panel and click Start
  2. Wait a few minutes: the panel shows live status
  3. When the IP appears, the server is ready

The server shuts down automatically when no players are online.

08

Join the Server

  1. Open the launcher, select the Fabric 1.20.1 profile
  2. In the main menu: Multiplayer → Direct Connection
  3. Paste the IP from the AriA Panel (e.g. 1.2.3.4:25565)
  4. Click Join Server and have fun!

Common Issues

ProblemSolution
Fabric won't openInstall Java (step 2)
Game crashes on launchDelete mods and re-extract from scratch
Fabric profile missingRedo step 3, version 1.20.1
Can't connectCheck the AriA Panel shows an IP
