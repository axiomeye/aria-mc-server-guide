Ciao! Questa guida ti aiuterà passo per passo a configurare Minecraft su Windows. Non serve essere esperti: ogni passaggio è spiegato nel dettaglio.

01

Installa SKLauncher e crea un'istanza Fabric

Il launcher è il programma che avvia Minecraft. Consigliamo SKLauncher perché permette di installare Fabric direttamente:

  1. Scarica SKLauncher da skmedix.pl e installalo
  2. Apri SKLauncher e clicca Add instance (il simbolo +)
  3. Seleziona Fabric come tipo e imposta la versione su 1.20.1
  4. Clicca Create, poi avvia l'istanza una volta. Chiudila subito appena compare il menu principale

Se hai già acquistato Minecraft e preferisci il Launcher Ufficiale, dovrai installare Java da adoptium.net e Fabric manualmente tramite il Fabric Installer (versione 1.20.1).

02

Installa le Mod

Le mod richieste dal server: senza di esse non ti farà connettere.

  1. In SKLauncher, fai clic destro sulla tua istanza Fabric 1.20.1 e seleziona Open Folder
  2. Se esiste già una cartella mods, eliminala completamente (clic destro → Elimina)
  3. Crea una nuova cartella vuota chiamata mods
  4. Scarica mods.zip dalla cartella condivisa (richiede account Google)
  5. Apri lo zip, seleziona tutto (Ctrl + A) e trascinalo nella cartella mods
03 · opzionale

Installa i Resource Pack

  1. Scarica resourcepacks.zip dalla cartella condivisa
  2. Apri la cartella dell'istanza (come al passo 2) e crea la cartella resourcepacks se non esiste
  3. Trascina tutto il contenuto dello zip nella cartella resourcepacks
  4. In gioco: Opzioni → Resource Pack, sposta i pack a destra per attivarli
04 · opzionale

Installa gli Shader

Richiedono un PC abbastanza potente. Se il tuo PC è datato, salta questo passaggio.

  1. Apri la cartella dell'istanza (come al passo 2) e metti Iris nella cartella mods
  2. Crea la cartella shaderpacks dentro la cartella dell'istanza
  3. Scarica Complementary Unbound e mettilo in shaderpacks
  4. In gioco: Opzioni → Shader, seleziona il pacchetto
05

Avvia il Server

  1. Apri l'AriA Panel e clicca Start
  2. Aspetta qualche minuto: il pannello mostra lo stato in tempo reale
  3. Quando compare l'IP il server è pronto

Il server si spegne in automatico quando non ci sono giocatori. Non preoccuparti di spegnerlo manualmente.

06

Collegati al Server

  1. In SKLauncher, seleziona la tua istanza Fabric 1.20.1 e clicca Play
  2. Nel menu principale: Multigiocatore → Connessione Diretta
  3. Incolla l'IP mostrato nell'AriA Panel (es. 1.2.3.4:25565)
  4. Clicca Connetti e buon divertimento!

Hey! This guide walks you through setting up Minecraft on Windows step by step. No experience required.

01

Install SKLauncher and create a Fabric instance

The launcher is the app that starts Minecraft. We recommend SKLauncher because it can install Fabric directly:

  1. Download SKLauncher from skmedix.pl and install it
  2. Open SKLauncher and click Add instance (the + button)
  3. Select Fabric as the type and set the version to 1.20.1
  4. Click Create, then launch the instance once. Close it as soon as the main menu appears

If you own Minecraft and prefer the Official Launcher, you'll need to install Java from adoptium.net and Fabric separately using the Fabric Installer (version 1.20.1).

02

Install the Mods

  1. In SKLauncher, right-click your Fabric 1.20.1 instance and select Open Folder
  2. If a mods folder exists, delete it entirely (right-click → Delete)
  3. Create a new empty mods folder
  4. Download mods.zip from the shared folder (Google account required)
  5. Open the zip, select all (Ctrl + A) and drag into mods
03 · optional

Install Resource Packs

  1. Download resourcepacks.zip from the shared folder
  2. Open the instance folder (same as step 2) and create a resourcepacks folder if it doesn't exist
  3. Drag the zip contents into it
  4. In-game: Options → Resource Packs, move packs to the right column
04 · optional

Install Shaders

Require a reasonably powerful PC. Skip if your PC is old.

  1. Open the instance folder (same as step 2) and place Iris in mods
  2. Create a shaderpacks folder inside the instance folder
  3. Download Complementary Unbound and place it in shaderpacks
  4. In-game: Options → Shaders, select the pack
05

Start the Server

  1. Open the AriA Panel and click Start
  2. Wait a few minutes: the panel shows live status
  3. When the IP appears, the server is ready

The server shuts down automatically when no players are online.

06

Join the Server

  1. In SKLauncher, select your Fabric 1.20.1 instance and click Play
  2. In the main menu: Multiplayer → Direct Connection
  3. Paste the IP from the AriA Panel (e.g. 1.2.3.4:25565)
  4. Click Join Server and have fun!

Common Issues

ProblemSolution
Game crashes on launchDelete mods and re-extract from scratch
Fabric instance missingRedo step 1 and make sure you selected Fabric version 1.20.1
Can't connectCheck the AriA Panel shows an IP
