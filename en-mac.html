Minecraft 1.20.1 Fabric ~15–20 min

Hey! This guide will walk you through setting up Minecraft on macOS step by step. No experience required — everything is explained.

01

Install a Minecraft Launcher

The launcher is the app that starts Minecraft. Pick one:

  • SKLauncher, recommended if you don't own Minecraft — download from skmedix.pl, choose the macOS version
  • Official Minecraft Launcher, if you have a Microsoft or Mojang account — download from minecraft.net

After installing, open the launcher, select version 1.20.1 and launch it once. You can close it as soon as the main menu appears — this just creates the game files.

02

Install Java

Java is a free program that Fabric needs to run.

Option A — Homebrew (if you have it):
Open Terminal and run: brew install --cask temurin

Option B — Manual download:
Go to adoptium.net, click Latest LTS release, open the file and follow the prompts.

If the Fabric installer opens without any error about Java, you can skip this step — you probably already have it.

03

Install Fabric

Fabric is a component that lets the game load mods. It's required to connect to the server.

  1. Download the Fabric Installer (it's a .jar file)
  2. Double-click the downloaded file to open it
    (If that doesn't work, open Terminal and run: java -jar ~/Downloads/fabric-installer-1.0.3.jar)
  3. In the window that appears, make sure the version is 1.20.1
  4. Click Install and wait a few seconds

If macOS blocks the file, go to System Settings → Privacy & Security and click Open Anyway.

04

Install the Mods

The server requires specific mods — without them you won't be able to connect.

First, open your Minecraft folder. Press Cmd + Space, type Finder, then in the menu bar go to Go → Go to Folder and paste: ~/Library/Application Support/minecraft

  1. If there's already a folder called mods, delete it entirely (right-click → Move to Trash)
  2. Create a new empty folder called mods
  3. Download mods.zip from the shared folder (requires a Microsoft account)
  4. Open the .zip, select everything (Cmd + A) and drag it all into the mods folder
05 — optional

Install Resource Packs

Resource packs change how the game looks. Not required, but they improve the experience.

  1. Download resourcepacks.zip from the shared folder
  2. Open your minecraft folder again (~/Library/Application Support/minecraft)
  3. Create a resourcepacks folder if it doesn't exist
  4. Open the zip and drag everything into minecraft/resourcepacks
  5. In-game, go to Options → Resource Packs and move packs from left to right to activate them
06 — optional

Install Shaders

Shaders make the game look much better, but require a reasonably powerful Mac. If your Mac is older, skip this.

  1. Download Iris and place it in the mods folder: iris-1.7.6+mc1.20.1.jar
  2. Create a shaderpacks folder inside the minecraft folder if it doesn't exist
  3. Download Complementary Unbound and place it in shaderpacks: ComplementaryUnbound_r5.5.1.zip
  4. In-game, go to Options → Shaders and select the shader pack
07

Start the Server

The server is started and stopped from the AriA Panel.

  1. Open the AriA Panel and click Start
  2. Wait a few minutes — the panel shows live status
  3. When the IP address appears, the server is up and you can connect

The server shuts down automatically when no players are online. Don't worry about turning it off manually.

08

Join the Server

  1. Open the launcher and select the Fabric 1.20.1 profile
  2. Launch the game
  3. In the main menu, click Multiplayer → Direct Connection
  4. Copy and paste the IP shown in the AriA Panel (example: 1.2.3.4:25565)
  5. Click Join Server and have fun!

Fabric profile not in launcherRedo step 3 — make sure you selected version 1.20.1
Can't connect to the serverCheck that the AriA Panel shows an IP — it may still be starting up
